Self-hostable

Run on your own infrastructure. Docker Compose for dev and production, with Postgres or SQLite.

Privacy-first

Single-tenant by design. Your data stays on your server — no telemetry, no third-party dependencies at runtime.

Plugin-first architecture

The plugin system is the product. Every capability — Console, Launcher, Account — is an installable plugin with a stable SDK contract.

Multi-user

Full role-based access control with owner and user roles, MFA (TOTP + passkeys), and invite-only registration.

Design system included

@sovereignfs/ui ships CSS custom property tokens and typed React components. Plugin developers get a consistent look for free.

Open source

AGPL-3.0. Fork it, self-host it, contribute back. The registry and plugin templates make it easy to build and share plugins.
